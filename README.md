# `py4etrics`

This module is for [Pythonで学ぶ入門計量経済学](https://py4etrics.github.io/)

### How to Use
```
import py4etrics 
```

### How to Install
```
pip install git+https://github.com/spring-haru/py4etrics.git
```
or
```
git clone https://github.com/spring-haru/py4etrics.git
cd py4etrics
pip install .
```
